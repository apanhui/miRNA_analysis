# miRNA_analysis

## Install 

Prepare software
+ blastall: v2.2.26
+ bowtie: v1.1.2
+ miRDeep2: v2.0.0.7
+ edgeR: v3.12.1
+ targetscan: v7.0
+ miranda: v3.3a

Prepare Database
+ Rfam: 11.0
+ mirbase_animal.fa: miRBase Release 22.1
+ mirbase_animal.txt: miRBase Release 22.1
+ hairpin_bta.fa: miRBase Release 22.1
+ mirbase_bta.txt: miRBase Release 22.1
+ genome.fa: Ensembl release 94(Bos taurus)
+ rRNA.fa: Ensembl release 94(Bos taurus)
+ 3UTR.fa: Ensembl release 94(Bos taurus)

## How to run
the main script is miRNA_analysis.sh, and must be run in Linux System, in bash windows, just type `sh miRNA_analysis.sh ` to run it.
